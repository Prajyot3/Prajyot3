### 🏆 Featured Project

#### 🧠 [Brain Tumor & Alzheimer's Detection System](https://github.com/Prajyot3/Brain-tumor-and-alzheimer)
An interactive web application powered by **Flask** and fine-tuned **EfficientNetB0 (TensorFlow/Keras)** deep learning models.
* **Brain Tumor Diagnostic**: Classifies MRI scans into Glioma, Meningioma, Pituitary, or No Tumor with $\ge 95\%$ validation accuracy.
* **Alzheimer's Diagnostic**: Analyzes scans to classify cognitive levels into Mild Demented, Moderate Demented, Very Mild Demented, or Non-Demented with $\ge 90\%$ validation accuracy.
* **Telehealth Chat Engine**: Real-time consultation portal built for Patients and Specialists.
* **Tech Stack**: Python, Flask, TensorFlow, OpenCV, SQLite, HTML5, CSS3, Bootstrap.
